This project performs sentiment analysis on restaurant reviews using Natural Language Processing (NLP) and Machine Learning techniques.

The goal is to classify customer reviews as **positive** or **negative** to better understand customer feedback.

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- NLTK
- Jupyter Notebook

## Project Workflow

1. Data preprocessing and cleaning
2. Text tokenization and feature extraction
3. Model training using machine learning algorithms
4. Sentiment classification of restaurant reviews

## Files in this Repository

- `Restaurant_sentiment_analysis.ipynb` → Main notebook containing the model and analysis
- `README.md` → Project documentation
